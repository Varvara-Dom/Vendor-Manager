# Vendor Manager

Windows Forms приложение для управления базой данных поставщиков (вендоров) и IP/MAC адресами.

## Описание

Приложение предоставляет функционал для:
- Управления справочником вендоров (добавление, редактирование, удаление)
- Редактирования таблицы IP/MAC адресов
- Автоматического определения вендора по MAC адресу
- Валидации формата MAC адресов

## Технологии

- .NET Framework 4.8
- Windows Forms
- Entity Framework 6
- MS SQL Server
- ADO.NET

## Структура базы данных

### Таблица gmc_Vendors
- `id` - уникальный идентификатор
- `macs` - MAC адрес вендора (формат: XX:XX:XX)
- `brand` - наименование вендора

### Таблица gmc_IPMAC
- `Id` - уникальный идентификатор
- `Mac` - полный MAC адрес устройства
- `ip_cur` - текущий IP адрес
- `Inbase` - флаг активности
- `last_dateupdate` - дата последнего обновления
